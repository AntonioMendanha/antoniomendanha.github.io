Aprendendo Javascript

Tríade da Web

Escrito por: Antonio Mendanha - 20/07/2021

A tríade da WEB é composta por: HTML - CSS - Javascript

De forma lúdica podemos comparar com o corpo humano, o HTML seria o nosso esqueleto, o CSS as roupas que vestimos, e o Javascript podemos considerar como os músculos e o cérebro.

Olhando por esse lado você ja entende o quanto é necessário ter conhecimento sobre os 3, mas saiba que você pode começar sem o javascript por exemplo. Para páginas estáticas, que não precisam de muita interatividade com o usuário você consegue manter durante um tempo uma página sem o uso do javascript. Ao longo da evolução de seu projeto você irá se deparar com o mesmo código sendo digitado várias vezes, como um cabeçalho, um rodapé, barras de navegação, etc. A partir desse momento já vale começar a pensar em ferramentas para te ajudar a organizar e dar a manutenção devida para o código.

Apesar de conhecer um pouco sobre o Javascript por causa dos projetos criados em React com a Rocketseat e com a Alura, eu dei um passo para trás para entender o principio de como funciona a linguaguem. A primeira coisa positiva foi que esse aprendizado me permitiu logo de cara corrigir alguns scripts que estava usando no Google Sheets. Entendi o uso do MAP e começei a utilizar ele da forma correta. Também reforçei meu conhecimento sobre o FILTER e REDUCE. Esses métodos são muito úteis para a manipulação de arrays, que eu utilizo muito no Google SHEETS.

Agora estou reforçando o entendimento sobre os métodos que no meu ponto de vista são os mais importantes para a web, WINDOW.DOCUMENT. São diversas as possibilidades, criar um conteudo para um tag HTML, alterar o contéudo, adicionar um tag HTML ao seu arquivo. Evoluir o tratamento dos formulários tanto na forma de submeter como na aplicação do mesmo dentro da própria página.
Pretendo em um outro momento criar artigos sobre esse aprendizado, como um treinamento mesmo. O material já está semi-preparado, só falta formatar e publicar. Esse conteúdo que criei está bem bacana e acho que vão gostar.
