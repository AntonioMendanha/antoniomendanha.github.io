Apanhando pro GITHUB e conhecendo o Gatsby

Sem tempo pra pensar em uma frase pra colocar aqui!

Escrito por: Antonio Mendanha - 31/08/2020

Minhas últimas semanas desde o último post foram muito puxadas. Não tive tempo para pesquisar nada sobre o problema que tive com o último reposítório no GitHub. No projeto da NLW#2 a pasta mobile está subindo vazia para o GitHub e em rápidas pesquisas não achei o que eu deveria fazer e como corrigir. Até testei alguns comandos, mas no final sem êxito. Devo ter apagado pelo menos umas 5 vezes o repositório. Enfim verifiquei alguns cursos focados em GITHUB justamente para poder me aperfeiçoar nesse sentido e poder corrigir esse erro e subir o projeto corretamente. Também não tive como fazer os aprimoramentos propostos no final da NLW#2 pra levar o Proofy para a versão 2.0 (mas um dia o negócio vai).

Tirando essa leve insatisfação, decidi que era hora de corrigir a interface para as postagens. A página continuará a ser estática apesar de continuar tendo as postagens do blog. Se no futuro não der mais a gente refatora e modifica, mas agora o foco está na agilidade para eu criar os conteúdos evitando erros. Vou utilizar os conceitos que já aprendi referente a SPA, pra quem não entende do assunto não tem nada a ver com aquele local pra emagrecer e/ou descansar. Single Page Application torna a navegação mais rápida e consequentemente como vou utilizar os mesmos componentes para todas as postagens também vou ganhar tempo e diminuir possivéis erros na criação do Html. Verifiquei que uma boa saída e utilizar o framework do Gatsby. Já iniciei os meus estudos no framework para em breve fazer essas alterações.

Quer arriscar saber quanto tempo vou demorar para alterar o site e fazer a próxima postagem?
Me manda lá no twitter e vamos ver se alguém acerta.
