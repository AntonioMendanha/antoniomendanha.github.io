Bem vindo JSON

Escrito por: Antonio Mendanha - 03/05/2021

Nas últimas semanas participei de alguns treinamentos online (NLW - Rocketseat e Imersão-Alura), esses treinamentos não apresentam muito o que eu estou fazendo com esse blog, mas a cada aula vendo as organizações de códigos e a refatoração vem a necessidade de organizar um pouco melhor o meu código.

A sequência dos itens que preciso refatorar nesta primeira etapa estão claras para mim.

  • Começar a usar o Javascritp
  • Criar JSON com os dados dos posts
  • Modificar estrutura da página principal do Blog-Dev com uma estrutura de laço e usar o JSON
  • Usar o conceito de componentes para o Menu e o Footer

Sei que vocês sabem, mas vamos do início. JSON - JavaScript Object Notation - um Anotação em formato de Objeto JavaScript. Como em um arquivo de excel que geralmente tabula os dados utilizando um arquivo CSV, o JSON pode ser usado da mesma forma. No meu exemplo, tenho uma estrutura para a criação dos post que é de repetição: Nome do post, data, criador, imagem, etc. Organizando o arquivo com essas anotações eu posso chamar o objeto no html da página Blog-Dev utilizando a estrutura de repetição. Assim o código fica mais limpo e escalável. Depois de implantar esse JSON e a função chamando uma estrutura de laço mapeando o JSON, não vou precisar mais mexer nessa estrutura a cada novo post. A única coisa que terei que fazer é depois de criar o html do novo post eu tenho que adicionar as informações do novo post no JSON, e a página será atualizada automaticamente, com a função sendo rodada.

Com o pouco conhecimento que já adquiri de estrutura de banco de dados, foi fácil montar a estrutura do JSON. Deixando bem limpo e fácil. Nas práticas que tenho visto, geralmente após a abertura das chaves todo mundo pula para a próxima linha e eu preferi manter o id junta da abertura do objeto para facilitar minha busca quando eu minimizar conforme a imagem que coloquei no topo do post.

Como o JSON já foi criado, agora estou na parte de criar o primeiro script chamando uma função para apresentar os resultados do JSON na página principal. Nessa mudança, a função deve incluir em torno de 6 linhas de código que irão eliminar em torno de 130 linhas que hoje tem no meu html apenas pra dificultar a manutenção do código.
