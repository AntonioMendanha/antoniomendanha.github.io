Projeto Imersão React

Obrigado pelo conteúdo ALURA

Escrito por: Antonio Mendanha - 03/08/2020

A última semana não fiz nenhuma atualização pois foquei na Imersão React proposta pela ALURA. Pra quem não conhece a Imersão é um projeto da ALURA, que visa em uma semana ensinar aos alunos começando um projeto do zero até o deploy da aplicação. Essa Imersão foi a Imersão REACT e o projeto foi o ALURAFLIX, um clone do Netflix.

Você pode conferir o resultado do meu projeto nesse link aqui.

Nessa Imersão teve muito JavaScript com o REACT e um pouco de CSS, alias um conceito novo pra mim "Tag Template String" e algo que deixou a cabeça um pouco confusa que foi o CSS misturado com o HTML no meio do JavaScript. Essa parte foi um pouco confusa, mas deu pra praticar bastante a utilização de bibliotecas, SPA (Single Page Application) e dois serviços diferentes: Vercel e Heroku. O Vercell é uma opção de deploy da sua aplicação direto do seu repositório do GitHub bem similar ao que temos no GitHub Pages pelo que vi e o Heroku é a opção de ter um servidor pra sua aplicação também puxando direto do seu repositório lá do GitHub.

Link do repositório do GitHub: ReactFlix - KCafe
