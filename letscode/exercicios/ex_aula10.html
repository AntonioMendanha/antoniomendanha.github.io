Exercícios de Javascript - Condicionais

Procurando o indice de uma letra com For

Crie uma função que receba 2 argumentos um texto e uma letra.
A funçao deverá percorrer o texto e descobrir o indíce que se encontra a letra passada.

Criando uma sequência de Fibonacci

Crie uma função que receba um argumento que será a quantidade de numeros da sequência de Fibonnaci
A função retornará o array que foi iniciado com [0, 1]

Soma condicional de um array

Crie uma função que receba um array
A função deverá retornar a soma dos números pares desse array.

Listar somente um valor de propriedades do objeto dentro de um array

Crie uma função que receba um array de funcionários
Formato do array de objetos: [{nome:'nomePrimeiro', idade: 22]
A função deverá retornar somente os nomes dos funcionários.
